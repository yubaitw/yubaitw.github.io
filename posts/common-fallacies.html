訴諸權威、滑坡、以人廢言——常見的邏輯謬誤解釋

滑坡謬誤

（這絕對不是真人真事）
A：「我去學校練吉他哦！」
B：「你再玩社團下去就考不上大學了！！！！」

B 的論證大致如下：

玩社團 -> 沒有時間讀書 -> 成績變爛 -> 學測考不好 -> 考不上大學

這一串事件構成了一個「滑坡」——每個事件都會導致下個事件發生。

但 B 的邏輯中的事件，不一定會導致下個事件發生（A 玩社團不代表 A 會沒時間念書）。

訴諸人身

訴諸權威

A：「我覺得 C 的投資方法不一定正確、、、」
B：「怎麼可能？他那麼有錢欸！！」

C 很有錢不代表他的投資方法（就算 C 真的靠投資變有錢，也不代表他的方法是對的）正確——可能他靠著寫投資書、繼承老爸的遺產變得富有，而非透過投資。也有可能 C 是倖存者偏差中的倖存者，使用錯誤的方法得到正確的結果。

但這不代表我們不能完全相信權威的話。訴諸權威謬誤成立的前提是 權威所說言論的領域並非他的專業。

醫生當然能提出感冒的預防方法，但他並沒有足夠的公信力提出「神是存在的」。

以人廢言

A：「節食不是減肥的最好的方法。」 B：「你這麼胖，照你說的做又怎麼可能減肥成功？」

「A 胖不胖」和「節食不是最好減肥方法」兩件事並無關聯。即使 A 很胖，他的主張依然可能正確。

校系歧視也可歸到此類。

訴諸未知

A：「我們無法確定上帝存在與否。」
B：「你沒辦法證明上帝不存在，所以上帝存在！」

B 的論證爲：「沒有證據顯示上帝不存在 -> 上帝存在」。但「沒有證據」並不能導出「上帝存在」或「上帝不存在」的結論。

碰上 B 這種人，有個簡單的反駁方法：

A：「你欠我一百萬。」
B：「你沒有證據證明你欠我錢！」
A：「你也沒有證據證明你沒欠我一百萬，所以你欠我一百萬！」

稻草人謬誤

A：「我認爲應立法管制槍支。槍支暴力太過氾濫了。」
B：「你反對無辜的人拿武器對付歹徒嗎？」

除了槍以外，無辜的人也能使用刀子、雨傘對付歹徒。B 的反駁和 A 的主張並不衝突。

A 也沒說過「我反對使用武器自衛」，但 B 曲解 A 的觀點（就像立一個稻草人攻擊）並加以攻擊。
